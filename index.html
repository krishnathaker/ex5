Backstretch Plug-in



Backstretch is a simple jQuery plugin that allows you to add a dynamically-resized, slideshow-capable background image to any page or element.


Other than a background slideshow, Backstretch also will stretch any image to fit the page or block-level element, and will automatically resize as the window or element size changes.


On this web page, I have utilized photographs from some cities of the British Isles as a background slideshow.
